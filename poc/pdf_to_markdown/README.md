# 阶段一：多模态模型 PDF 解析能力测试方案

## 测试目的
验证各个大模型（特别是 Gemini 1.5 Pro 和最新的 2.0/2.5/3.1 Pro 系列）在处理初中理科（包含大量 LaTeX 公式、几何图形、物理受力图）试卷时的精准度。这直接决定了我们系统的“数据预处理 Pipeline”设计方案。

## 前置准备工作
1. **安装环境依赖**：
   在终端运行：`pip install google-genai`
2. **准备测试物料**：
   从网上下载一份典型的《初中八年级下册数学期末试卷.pdf》并命名为 `sample_test.pdf`，放入此目录下。
3. **配置密钥**：
   在终端中配置您的 API KEY：
   `export GEMINI_API_KEY="您的_API_KEY_在这里"`

## 评测执行
运行脚本：
```bash
python test_gemini.py
```
*此脚本会自动上传 PDF，分别调用不同版本的模型进行结构化抽取，并将结果输出为 `result_*.md` 文件。*

## 核心人工评估指标 (需在生成的 md 文件中检查)
当你得到输出后，请重点评估以下三项：
1. **公式还原度**：检查根号、分数、指数是否被正确转化为 LaTeX 代码。
2. **“Vision Adaptor” 的图转文能力**：重点观察几何题，看 AI 是否能在题干中，把原图的“辅助线、垂直关系”用文字极其详尽地描写了出来。
3. **分步解析逻辑**：检查高级模型给出的大题步骤是否符合初中生的思考逻辑，是否真的做到了“细粒度拆解”。
