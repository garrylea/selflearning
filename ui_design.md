# 前端交互设计说明书 (UI/UX Design Specification)

## 1. 整体视觉逻辑
本 App 旨在为初中生提供一个**“探索式”**的学习体验。界面不再是枯燥的列表，而是一个可以互动的**知识宇宙**。

### 1.1 核心视图：知识宇宙 (The Map)
- **形态**: 全屏动态拓扑图。
- **交互**: 鼠标滚轮缩放，左键拖拽。
- **节点语义**:
    - **锁定态 (Lock)**: 灰色，点击提示“请先攻克前置知识点：xxx”。
    - **激活态 (Active)**: 浅色边缘呼吸灯效果，表示当前建议练习。
    - **精通态 (Mastered)**: 深色实心高亮，表示已通过 A 级挑战。
- **Agent 精髓**: 当用户停留在地图上超过 5 秒，Agent 会在角落弹出一个气泡提示：“探测到你的‘相似三角形’知识点存在薄弱环节，是否现在进入训练？”

---

## 2. 核心交互流程拆解

### 2.1 知识点开启流
1. 用户在 **Map 视图** 点击节点。
2. 右侧滑出 **知识卡片 (Knowledge Card)**。
3. 展示：名称、Markdown 定义、学习意义。
4. 点击“进入练习”，界面进入 **沉浸式训练模式**。

### 2.2 沉浸式训练模式 (Immersive Practice)
- **顶部**: 掌握度进度条（例如：Level C [████░░] 2/3）。
- **中心**: 题目区域。
    - **核心渲染要求**: 集成 **KaTeX**，确保题目、解析中的几何符号（∠, △, ⊥, ∥）和公式符号排版完全符合初中数学教材规范。
    - 支持图形描述 Blockquote 展开，公式与文字对齐。
- **底部**: 交互 Dock 栏 (The Multi-modal Input Dock)。
    - **拍照提交 (Primary)**: 针对手写解题的学生，提供显眼的“拍照/上传”按钮，支持手机端直接拍照或桌面端截图上传。
    - **Input**: 供熟悉 LaTeX 的学生输入或预览最终结果。
    - **Audio**: 按住录音，松开发送提问。
- **反馈**: 答对时弹出绿色即时反馈；答错时，右侧滑出 **AI 助教面板**。

### 2.3 AI 助教与多媒体会话 (AI Tutor & Multimedia)
- **交互精髓**: 
    - 助教面板会自动引用题目中的具体行号。
    - **多媒体历史记录**: 采用时间轴（Timeline）展示。
    - **回放体验**: 
        - 音频消息显示波形图，点击直接在当前上下文播放。
        - 视频消息支持弹窗播放，并自动关联题目解析的特定步骤。

## 3. 关键组件规范

### 3.1 数学公式与 LaTeX 渲染引擎
- **引擎选型**: 优先使用 **KaTeX** 以获得最佳渲染性能。
- **渲染规范**:
    - **几何符号**: 角 (`\angle`)、垂直 (`\perp`)、平行 (`\parallel`)、三角形 (`\triangle`) 必须渲染成标准的几何符号。
    - **代数符号**: 分式、根号、希腊字母、指数/下标需符合规范排版。
- **预览机制**: 具备实时预览窗口，显示 LaTeX 渲染后的视觉效果。

### 3.2 练习评估报告
- **练习结束触发**：弹出全屏报告。
- **数据指标**：
    - **能力雷达图**：运算能力、空间想象、逻辑推理等。
    - **掌握度变动**：例如“恭喜，你的勾股定理掌握度从 D 升至 C！”。
    - **错题回溯**：点击错题直接进入 AI 对话模式复盘。

## 4. 技术实现映射
- **跨端外壳**: Tauri (macOS/Windows/Linux)。
- **图表**: AntV G6 (图谱) + ECharts (报表)。
- **渲染**: React 18 + Tailwind CSS (布局) + KaTeX (公式)。
- **实时性**: SSE (AI 响应) + WebSocket (状态同步)。
